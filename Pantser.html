Pantser

Panster is a 2D action rogue-like bullet hell game inspired by games like Vampire Survivors and Magic: Survival.
The game is currently in early development, this page will be updated when progress is made.

Upgrade system

When the player levels up they will be able to chose between a new ability or upgrade one they already had. To do this I wanted to let the player choose between 3 or more cards, which required a couple things. A way to generate the cards from scriptableobjects and something that keeps track of which abilites are active and what level they are.

Enemy Spawner

I had already created a way to quickly make new types of enemies but I needed a way to spawn them in. Specifically in a way where the game would get progressivly harder. To do this I made a function to decrease the spawn interval time and start spawning stronger types of enemies the longer the game goes on.

Object Pooling

One of the best ways to optimize bullet hell games is by using object pooling. I used it for both the enemy spawning and the projectiles the player will shoot.

Project description

Pantser is a project I started to see what I would/could make if there were no boundaries and to see how much I'm capable of as a developer. I specifically chose the bullet hell genre because those games require a lot of optimization and that's something I haven't had to do for other project but I feel like is a necessary skill to have as a game developer.
