Quantum Shuffle

Quantum shuffle is a first person puzzler game where the goal is to escape a mysterious laboratory. To do this you'll have to access a control panel where you'll find a grid with rooms on it. These rooms on the grid correspond directly to the rooms you'll walk through while trying to escape. Link up the doors to connect rooms to each other and line them up so they create a path to the exit. Check it out on Itch.io!

My Contributions:

Event System:

During development we noticed that a lot of interactions needed events to run or to communicate with a different behaviour. So I made this system to make it easier to quickly add new events to gameobjects. It works by having an event listener which checks the event it needs to play and link that with a key. This event listener automatically adds itself to the dictionary in the event manager. Finally, there is an event invoker which in our case was most often a collider. Learn more.

First Person Controller:

One of the requirements for this project was to make it in first person, so this was the first thing I made. I used both unity's new input system and the character controller to make this. The character controller is used to move the player although the move & look direction are calculated in my script. Learn more.

Project Description:

For this 8-week school project we had to make a first person puzzle game. The first two weeks were about pre-production, in this phase we had to come up with a main mechanic, present our ideas, design levels around this mechanic, and build a proof of concept. After that we started working on production for the next 6 weeks were the primary focus was on creating a polished game.
